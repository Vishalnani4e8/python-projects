# Power BI Sales Performance Dashboard

## Project Overview
This project presents an interactive Power BI dashboard to analyze sales performance.

## KPIs
- Total Sales
- Total Profit
- Total Orders
- Average Discount

## Visuals Used
- Bar chart (Sales by Category)
- Line chart (Monthly Sales Trend)
- Donut chart (Profit by Segment)
- Map (Sales by Region)

## Tools
- Power BI Desktop
- CSV Dataset

## Outcome
This dashboard helps identify trends, top-performing categories, and regional sales insights.
